The CSS Flexbox is a one-dimensional layout. It is useful in allocating and aligning the space among items in a grid container. The CSS Grid Layout is a two-dimensional grid-based layout system with rows and columns. It is useful in creating more complex and organized layouts. Flexbox takes a basis in the content while Grid takes a basis in the layout.

Bootstrap is a Component-based Framework. It's designed with a set of pre-made components, along with some utilities for spacing, color, column etc to help developer to build websites or app quickly and easily. On the other hand, tailwindCSS is a Utility-first Framework. Tailwind is all utilities. The core framework has no pre-made components. Bootstrap uses some javascript bundles to make tke components functional. Tailwind uses only css. and It has make css more user friendly.

The CSS Box Model is a term used for the div that wraps the following element properties within it. Box model help us to adjust an element's sizing. We can see this box model by inspecting any component of a webpage

  • Margin
  • Border
  • Padding
  • Content
  • Height and Width

Semantic elements have meaningful names which tells about type of content. HTML5 introduces many semantic tags, which make the code easier to write and understand for the developer and instructs the browser on how to treat them. Some examples are given below

  • aside
  • header
  • footer
  • section
